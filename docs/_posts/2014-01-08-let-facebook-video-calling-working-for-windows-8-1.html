Let me guess, you are upgraded your Windows 8 to Windows 8.1 and then Facebook video calling icon suddenly missing on Facebook. This happening because Windows 8.1 is not yet compatibility for Facebook video calling plugin.

In this post I will show you how you could fix this compatible problem, and bring video call icon back to Facebook.

This guide is working only for Google Chrome.

1. Explore to directory:
64-bit:

{% highlight shell %} C:\Program Files (x86)\Google\Chrome\Application {% endhighlight %}

32-bit:

{% highlight shell %} C:\Program Files\Google\Chrome\Application {% endhighlight %}

2. Right-click chrome.exe and select "Properties"
3. Select "Compatibility" -tab
4. Tap "Run this program for compatibility mode for" checkbox and choose Windows 8!
5. Click Ok
6. Open Google Chrome and enjoy!
