I am writing this post as part of course Linuxin keskitetty hallinta held by Tero Karvinen. In this post I will write hello world puppet module.

Install Puppet

{% highlight shell %} $ sudo apt-get update $ sudo apt-get install puppet {% endhighlight %}

Puppet oneliner

{% highlight shell %} $ puppet apply -e 'file { "/tmp/helloPuppet": content => "Hello Puppet!" }' {% endhighlight %}

Create new hello module

{% highlight shell %} $ mkdir -p modules/hello/manifests/ $ nano modules/hello/manifests/init.pp {% endhighlight %}

Do next class in file init.pp:

{% highlight shell %} class hello { file { '/tmp/helloPuppet': content => "Hello Puppet!\nPuppet module working as well!" } } {% endhighlight %}

Run puppet module:

{% highlight shell %} $ puppet apply --modulepath modules/ -e 'class {"hello":}' {% endhighlight %}

Last we verify that module works right:

{% highlight shell %} $ less /tmp/helloPuppet {% endhighlight %}

Result

Hello Puppet!
Puppet module working as well
